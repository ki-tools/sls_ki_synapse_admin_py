Create Synapse Space

Help

This process will do the following:

  • Create a new Synapse project.
  • Set the project to use an encrypted storage location.
  • Create a new Synapse team and add it to the project with edit/delete permission.
  • Add a specific set of users and/or teams to the project.
  • Invite all the email addresses to the team.
  • Create a default set of folders in the project.
  • Copy a wiki into the project from another Synapse project.
  • Update a Synapse table with details about the project that was created.
